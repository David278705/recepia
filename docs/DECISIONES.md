# Decisiones tomadas de forma autónoma

Cada vez que Claude Code decide algo sin poder consultar, lo registra aquí. El humano revisa esta lista al despertar y revierte lo que no le guste.

Formato:

## [YYYY-MM-DD] Título de la decisión
- **Contexto**: qué problema había.
- **Decisión**: qué se eligió.
- **Alternativas**: qué más se consideró y por qué se descartó.
- **Cómo revertir**: commit(s) o pasos.
