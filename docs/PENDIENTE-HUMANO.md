# Pendiente de decisión humana

Cosas que Claude Code NO puede decidir solo (criterio de negocio, dinero, marca, prioridades, acciones externas irreversibles). Cada punto debe poder resolverse leyéndolo en 30 segundos.

Formato:

## [YYYY-MM-DD] Pregunta concreta
- **Contexto**: lo mínimo necesario para entender.
- **Opciones**: A / B / C, con la implicación de cada una.
- **Recomendación**: cuál elegiría Claude Code y por qué.
- **Bloquea**: qué tarea del backlog queda detenida por esto (o "nada, solo informativo").
